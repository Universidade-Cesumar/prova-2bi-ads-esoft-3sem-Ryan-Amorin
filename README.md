## 📋Descrição

Este projeto foi desenvolvido para otimizar o controle de materiais em
almoxarifado do setor de saúde, um ambiente onde a agilidade e a precisão
são críticas. O sistema oferece uma solução clara para o gerenciamento de
estoque, permitindo o acompanhamento em tempo real das entradas e
saídas de itens. A aplicação consome a MockAPI para realizar operações de
CRUD, garantindo persistência de dados de forma ágil e demonstrando uma
arquitetura eficiente para o controle de materiais.

## ✨ Funcionalidades

**Cadastro do Material** -> adiciona item a tabela de materiais.
